# LARA 3D Immersion

## Local development

Requirements:

- Node.js
- npm

Run the app:

```sh
npm install
npm run dev
```

Default dev server URL:

- http://localhost:8080

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build production assets
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
