# Therapist Dashboard Specification

## Purpose

Provide structured insight into learning progress without exposing raw
emotional transcripts.

------------------------------------------------------------------------

## Dashboard Modules

1.  Educational Progress

-   Mastery Level per Concept
-   Attempt Count
-   Success Rate
-   Highest Achieved Level

2.  Emotional Stability Trends

-   Frustration Frequency per Concept
-   Recovery Count
-   Neutral Stability Ratio

3.  Session Summaries

-   Session Duration
-   Concepts Practiced
-   Difficulty Changes
-   Recovery Interventions Triggered

------------------------------------------------------------------------

## What Is Not Displayed

-   Raw transcripts
-   Emotional labels
-   Psychological interpretation
-   Diagnostic summaries

------------------------------------------------------------------------

## Safety Design

-   Metrics only
-   Aggregated counts
-   Decay-based emotional trends
-   No permanent tagging

------------------------------------------------------------------------

## Long-Term Value

Therapists can: - Identify trigger concepts - Monitor resilience
growth - Adjust intervention strategies - Validate LaRa's adaptation
behavior

The dashboard informs human care. It does not replace it.
